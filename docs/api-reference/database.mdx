## `vxdb.Database`

The entry point for embedded mode. Holds and manages multiple collections.

```python
import vxdb

# Persistent — data survives restarts
db = vxdb.Database(path="./my_data")

# In-memory — fast and ephemeral
db = vxdb.Database()
```

**Constructor parameters:**

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `path` | `str` | No | `None` | Directory for persistent storage. If omitted, the database is in-memory only. |

When `path` is provided:
- Collections, vectors, metadata, and documents are persisted to disk automatically on every write.
- Data is stored using **mmap** (vectors), **SQLite** (metadata + documents), and a **write-ahead log** (crash recovery).
- Reopening the same path restores all data.

When `path` is omitted:
- Everything lives in memory. Data is lost when the process exits.
- Ideal for prototyping, tests, and ephemeral workloads.

## Methods

### `create_collection`

Create a new collection.

```python
collection = db.create_collection(
    name="docs",
    dimension=384,
    metric="cosine",
    index="flat",
)
```

**Parameters:**

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `name` | `str` | Yes | — | Unique collection name |
| `dimension` | `int` | Yes | — | Number of floats per vector |
| `metric` | `str` | No | `"cosine"` | Distance metric: `"cosine"`, `"euclidean"`, `"dot"` |
| `index` | `str` | No | `"flat"` | Index type: `"flat"` (exact), `"hnsw"` (approximate) |

**Returns:** `Collection`

**Raises:** Error if a collection with that name already exists.

---

### `get_collection`

Get an existing collection by name.

```python
collection = db.get_collection("docs")
```

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `name` | `str` | Yes | Collection name |

**Returns:** `Collection`

**Raises:** Error if the collection does not exist.

---

### `list_collections`

List all collection names.

```python
names = db.list_collections()
# ["docs", "images"]
```

**Returns:** `list[str]`

---

### `delete_collection`

Delete a collection and all its data.

```python
db.delete_collection("docs")
```

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `name` | `str` | Yes | Collection name |

**Raises:** Error if the collection does not exist.
